# Emp-Info-Hub

 A simple and efficient **Employee Management System** to manage staff records via the console.

 ## Features
  **Data Entry:** Easy input for adding new staff.
   **Details Tracked:**
   - Employee Name
   - Age
   - Years of Experience
  **Console Output:** Instant display of all recorded information.

## Technologies Used
  **Language:** Java

### Technical Implementation

In this project, I have implemented several core java concepts:

* **Constructors:** Used **Parameterized Constructors** to initialize employee objects with specific data (Name, Id, Age, Experince) at the moment of creation.
* **Arrays:** Utilized **Arrays** to store multiple employee records, enabling the system to manage a collection of data and display them efficiently using loops.
* **Exception Handling:** Integrated **try-catch blocks** to handle potential input errors (Like entering text where a number is expected), ensuring the program dones't crash during execution.
